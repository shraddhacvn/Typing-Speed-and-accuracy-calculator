# Typing Speed and Accuracy Calculator

**Typing Speed and Accuracy Calculator** is a Python application developed using Tkinter for the GUI. This application helps users measure their typing speed and accuracy by providing random sentences or paragraphs for the user to type.

## Features

- **User Registration**: Users can enter their name to start the test.
- **Random Sentences**: The application provides random sentences or paragraphs for the test.
- **Typing Speed Calculation**: Calculates and displays typing speed in words per minute (WPM).
- **Accuracy Calculation**: Calculates and displays the accuracy percentage.
- **Record Stats**: Saves test statistics including date, time, user name, typing speed, and accuracy to a file.

## Technologies Used

- **GUI**: Tkinter
- **Backend**: Python
- **Data Storage**: Local file system (records.txt)

## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/shraddhacvn/typing-speed-and-accuracy-calculator.git
    ```
2. Navigate to the project directory:
    ```bash
    cd typing-speed-and-accuracy-calculator
    ```
3. Ensure you have Python installed. You can download it from [python.org](https://www.python.org/).
4. Run the application:
    ```bash
    main.py
    ```

## Usage

1. Run the application by executing `main.py`.
2. Enter your name in the provided field and click "Go" to start the test.
3. A new window will open with a random sentence or paragraph displayed.
4. Start typing the sentence or paragraph in the provided text area.
5. The application will calculate and display your typing speed (WPM) and accuracy.
6. You can save your test statistics by clicking the "Record" button.
7. Click "Try Again" to restart the test with a new sentence or paragraph.

## File Descriptions

- **main.py**: The main script to run the application.
- **records.txt**: File where test statistics are recorded.
- **sentences.txt**: File containing sentences or paragraphs used for the typing test.
